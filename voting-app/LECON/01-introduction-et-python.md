# Étape 1 : Le Langage Python

## Qu'est-ce que c'est ?
Python est un langage de programmation interprété, très populaire, particulièrement utilisé pour le développement web, l'intelligence artificielle et l'automatisation (l'approche DevOps).

## Le choix technique dans ce projet
L'application "Azure Voting App" est écrite en Python. Le fichier `README.md` indique qu'il faut utiliser au minimum la version **3.13**. 
L'intérêt technique de spécifier une version est de s'assurer que les fonctionnalités utilisées dans le code (qui peuvent évoluer et changer d'une version à l'autre) soient bien supportées, garantissant ainsi que l'application s'exécute sans erreur de compatibilité.

## Comment ça fonctionne ?
Pour exécuter le code de l'application (le fichier `main.py`), on utilise un programme appelé l'**interpréteur Python**, qui lit le code ligne par ligne et l'exécute directement sur la machine sans avoir besoin de le compiler à l'avance en langage machine.
